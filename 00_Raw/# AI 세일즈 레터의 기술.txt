# AI 세일즈 레터의 기술

## 제1장: AI 시대의 세일즈 레터
1. 왜 지금 AI 세일즈 레터인가
   - 전통적 세일즈 레터의 한계
   - AI가 가져온 변화와 기회
   - ChatGPT AI의 특장점

2. 세일즈 레터의 진화와 현재
   - 전통적 DM에서 디지털 시대로
   - 멀티채널 세일즈 레터의 부상
   - AI 시대의 새로운 가능성

3. 세일즈 레터의 활용 분야
   - 이커머스와 랜딩페이지
   - 이메일 마케팅
   - 소셜미디어 광고 카피
   - 제품 상세 페이지
   - 회원가입/구독 유도
   - 브랜드 스토리텔링

4. AI 세일즈 레터의 기본 원리
   - 설득의 심리학과 AI
   - 데이터 기반 카피라이팅
   - 고객 심리 모델링

## 제2장: ChatGPT AI 마스터하기
1. ChatGPT AI 기본기 다지기
   - 계정 설정과 초기 환경
   - 기본 프롬프트 작성법 
   - 프롬프트 체이닝 기법

2. 세일즈 레터 특화 프롬프트 패턴
   - 페르소나 분석 프롬프트
   - 가치 제안 도출 프롬프트
   - 감정 공감대 형성 프롬프트
   - 설득력 강화 프롬프트

3. 프롬프트 최적화 전략
   - 컨텍스트 제공 방법
   - 제약조건 설정 기법
   - 피드백 루프 구축
   - 결과물 품질 관리

## 제3장: 고객 심리 분석과 타겟팅
1. AI로 시작하는 고객 분석
   - 타겟 페르소나 생성 프롬프트
   - 잠재 고객 니즈 발굴 프롬프트
   - 구매 여정 매핑 프롬프트

2. 심리 트리거 포인트 발굴
   - 페인포인트 분석 프롬프트
   - 구매 동기 파악 프롬프트
   - 저항점 극복 프롬프트

3. 맞춤형 가치 제안 설계
   - USP 도출 프롬프트
   - 차별화 포인트 발굴 프롬프트
   - 경쟁사 분석 프롬프트

## 제4장: AI 세일즈 레터 작성의 실제
1. 강력한 헤드라인 작성
   - 주의환기 헤드라인 프롬프트
   - 호기심 유발 패턴
   - A/B 테스트용 변형 생성

2. 도입부 최적화
   - 공감대 형성 프롬프트
   - 스토리텔링 전개 프롬프트
   - 관심 지속 프롬프트

3. 본문 구조화
   - AIDA 구조 최적화 프롬프트
   - PAS 프레임워크 적용 프롬프트
   - 증거와 사회적 증명 강화 프롬프트

4. 행동 유도 (CTA) 설계
   - 긴급성 부여 프롬프트
   - 혜택 강조 프롬프트
   - 저항 해소 프롬프트

## 제5장: 상황별 세일즈 레터 최적화
1. 신규 고객 유치
   - 첫인상 강화 프롬프트
   - 신뢰도 구축 프롬프트
   - 리스크 제거 프롬프트

2. 기존 고객 업셀
   - 로열티 활용 프롬프트
   - 추가 가치 제안 프롬프트
   - 구매 이력 연계 프롬프트

3. 이탈 고객 재유치
   - 재연결 메시지 프롬프트
   - 개선점 피드백 프롬프트
   - 특별 혜택 제안 프롬프트

## 제6장: 성과 측정과 지속적 개선
1. A/B 테스트 설계
   - 테스트 요소 선정 프롬프트
   - 변형 생성 프롬프트
   - 결과 분석 프롬프트

2. 데이터 기반 최적화
   - 성과 지표 분석 프롬프트
   - 개선점 도출 프롬프트
   - ROI 계산 프롬프트

3. 반복 가능한 성공 시스템 구축
   - 워크플로우 자동화 프롬프트
   - 품질 관리 체크리스트
   - 템플릿 최적화 프로세스

## 제7장: 채널별 세일즈 레터 최적화
1. 이메일 마케팅 최적화
   - 제목 최적화 프롬프트
   - 본문 구조화 프롬프트
   - 시퀀스 설계 프롬프트
   - 스팸 필터 회피 프롬프트

2. 랜딩페이지 최적화
   - 페이지 구조 설계 프롬프트
   - 스크롤 패턴 분석 프롬프트
   - 전환율 최적화 프롬프트
   - UX 라이팅 프롬프트

3. 소셜미디어 광고 카피
   - 플랫폼별 톤앤매너 프롬프트
   - 캐러셀 광고 시나리오 프롬프트
   - 해시태그 최적화 프롬프트
   - 인게이지먼트 향상 프롬프트

4. 제품 상세 페이지
   - 상품 스토리텔링 프롬프트
   - 특장점 부각 프롬프트
   - 구매 불안 해소 프롬프트
   - 리뷰 활용 프롬프트

## 제8장: ROI 극대화 전략
1. 전환율 최적화
   - 전환 장벽 분석 프롬프트
   - 마찰 포인트 제거 프롬프트
   - 구매 여정 최적화 프롬프트

2. 비용 효율성 향상
   - 제작 시간 단축 전략
   - 리소스 최적화 프롬프트
   - 스케일링 자동화 프롬프트

3. 투자 대비 성과 측정
   - 성과 지표 설정 프롬프트
   - 비용 분석 프롬프트
   - ROI 추적 프롬프트

4. 성공 사례 분석
   - 업종별 성공 사례
   - 투자금액 대비 매출 증대
   - 전환율 개선 사례
   - 비용 절감 효과

## 제9장: 세일즈 레터 기반 마케팅 전략 수립
1. 마케팅 인사이트 도출
   - 고객 반응 분석 프롬프트
   - 경쟁력 분석 프롬프트
   - 시장 기회 발굴 프롬프트

2. 통합 마케팅 전략 설계
   - 채널 믹스 최적화 프롬프트
   - 캠페인 기획 프롬프트
   - 콘텐츠 전략 수립 프롬프트

3. 마케팅 실행 계획 수립
   - 실행 일정 설계 프롬프트
   - 예산 계획 프롬프트
   - KPI 설정 프롬프트

4. 자동화와 스케일링
   - 마케팅 자동화 프롬프트
   - 데이터 수집 자동화 프롬프트
   - 최적화 자동화 프롬프트

5. 성과 분석과 전략 조정
   - 성과 분석 프롬프트
   - 개선점 도출 프롬프트
   - 전략 조정 프롬프트
